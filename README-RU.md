# Focus Blocker

Расширение для браузера, которое блокирует обработчики событий фокуса и изменения видимости страницы на всех веб-сайтах, давая вам контроль над тем, когда сайты могут обнаруживать вашу активность.

## Возможности

- Глобальная блокировка событий фокуса и видимости
- Настраиваемый черный список доменов, где расширение должно быть отключено
- Инструмент Copy Helper для удобного выделения и копирования текста
- Простой интерфейс переключения для быстрого включения/выключения
- Настраиваемые параметры блокировки событий

## Установка

1. Клонируйте этот репозиторий
2. Откройте страницу управления расширениями вашего браузера
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное" и выберите директорию проекта

## Использование

Нажмите на иконку расширения, чтобы включить/выключить его. Используйте кнопку "Настройки" для настройки блокируемых событий и управления черным списком доменов.

Для подробной информации см. [INFO-RU.md](INFO-RU.md).
