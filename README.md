# Focus Blocker

A browser extension that blocks focus event listeners and page visibility changes on all websites, giving you control over when websites can detect your activity.

## Features

- Blocks focus and visibility events globally
- Configurable blacklist for domains where the extension should be disabled
- Copy Helper tool for easy text selection and copying
- Simple toggle interface for quick enable/disable
- Customizable event blocking settings

## Installation

1. Clone this repository
2. Open your browser's extension management page
3. Enable "Developer mode"
4. Click "Load unpacked" and select the project directory

## Usage

Click the extension icon to toggle it on/off. Use the Settings button to configure which events to block and manage your domain blacklist.

For detailed information, see [INFO.md](INFO.md).
